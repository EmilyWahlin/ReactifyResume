# How to clone my git:

You need to have node installed:

npm install -g node

Clone the repository:

git clone https://github.com/EmilyWahlin/ReactifyResume.git

Change into the project directory:

cd your-repo

Install dependencies:

npm install

Start the development server:

npm run dev

npm run start
